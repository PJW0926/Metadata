A. Opening Narrative

도메인: educational digital games
이 분야에서 메타데이터가 왜 필요한가: 게임에도 교육적 기능이 있는데 배척되기 쉬우므로 체계적인 메타데이터를 짜면 교육 현장에서 사용하기 좋지 않을까
주요 이용자: 교사, 연구자, 학부모, 사서
Draft 1-2의 주제는 병원, 의학 정보센터에 맞는 주제였으나 전문성이 강해 주제 변경.
Draft 3-4의 주제는 교육 현장에서 사용 가능한 게임 스키마.
스키마 설계에서 중요하게 생각한 점: 도메인 특화 요소
